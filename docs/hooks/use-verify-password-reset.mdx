## What it does

`useVerifyPasswordReset` manages the reset verification form, resend behavior, cooldown handling, and redirect behavior after the new password is submitted.

## Why it exists

The password reset verify screen has distinct flow logic but still builds on the same shared auth form pattern. This hook keeps those responsibilities grouped together.

## Used by

- `VerifyPasswordResetForm`
