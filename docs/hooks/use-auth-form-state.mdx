## What it does

`useAuthFormState` creates the shared form controller, validation wiring, local loading state, global error state, and field config lookup for an auth form.

## Why it exists

Most auth screens need the same form plumbing. This hook keeps that wiring in one place so the screen hooks can stay focused on submit behavior.

## Used by

- `useAuthForm`
- `useRequestPasswordReset`
- `useVerifyEmail`
- `useVerifyPasswordReset`
