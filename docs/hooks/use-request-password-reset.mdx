## What it does

`useRequestPasswordReset` owns the request-reset form state, runs the send-code action, and redirects the user into the verify-reset step on success.

## Why it exists

The request-reset screen has its own flow behavior, but it still depends on the shared auth form plumbing. This hook keeps that screen-specific behavior out of the render tree.

## Used by

- `RequestPasswordResetForm`
