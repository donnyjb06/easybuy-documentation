## What it does

`useAuthForm` coordinates submit behavior for the shared login and registration screen, including auth calls and redirect behavior.

## Why it exists

`AuthForm` should stay mostly declarative. This hook carries the branch logic that decides whether the current screen is handling login or registration.

## Used by

- `AuthForm`
