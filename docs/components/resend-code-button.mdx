## What it does

`ResendCodeButton` either shows the remaining cooldown time or renders the resend action once the cooldown has expired.

## Why it exists

Verification and recovery screens both need the same resend behavior. This component keeps that state presentation tiny and consistent.

## Example

```tsx
<ResendCodeButton
  isRunning={isRunning}
  seconds={seconds}
  handleResendCode={handleResendCode}
/>
```

## Props

| Prop | Type | Notes |
| --- | --- | --- |
| `isRunning` | `boolean` | Whether the cooldown is still active. |
| `seconds` | `number` | Remaining cooldown seconds. |
| `handleResendCode` | `() => Promise<void>` | Async resend callback. |
