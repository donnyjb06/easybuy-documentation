## Core Commands

```bash
# frontend
cd C:\Dev\Repos\easybuy-frontend\vendor
npm install
npm run start
npm run storybook
npm run docs:api

# docs site
cd C:\Dev\Repos\easybuy-documentation
npm install
npm run start
npm run build
```
