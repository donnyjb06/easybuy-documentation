## Install And Run The Frontend

```bash
cd C:\Dev\Repos\easybuy-frontend\vendor
npm install
npm start
```

Once the Expo dev server is running:

- press <code>w</code> for the web build
- press <code>a</code> for Android
- press <code>i</code> for iOS on macOS
- or scan the QR code in Expo Go on a physical device

You can also use the explicit scripts:

```bash
cd C:\Dev\Repos\easybuy-frontend\vendor
npm run web
npm run android
npm run ios
```

## Docs Workflow After The App Runs

```bash
# Storybook source and static build
cd C:\Dev\Repos\easybuy-frontend\vendor
npm run storybook
npm run build-storybook

# API reference generation
cd C:\Dev\Repos\easybuy-frontend\vendor
npm run docs:api

# Docs site
cd C:\Dev\Repos\easybuy-documentation
npm install
npm start
```
